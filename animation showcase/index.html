Realistic Calculator

A calculator with various features. The calculator has a power button to turn it on and off, a display for showing the input and results, and a set of numeric and operator keys for performing mathematical calculations. It also provides functionality for basic operations such as addition, subtraction, multiplication, division, square root, and more. The JavaScript code handles user interactions and calculations when the user clicks on the keys. The calculator can be turned on and off by clicking the power button, and it offers a user-friendly interface for performing arithmetic calculations.

Physics Demo

A 3D scene using the Three.js library and a physics engine using the Cannon.js library. It creates a ground plane, adds multiple colored spheres with varying sizes and physical properties to the scene, and simulates their physics, including gravity, collisions, and lighting effects. Users can control the camera's movement using arrow keys. The simulation renders the scene, updating the positions and orientations of the spheres over time.

Planets Demo

This showcases a 3D planetary environment created using the Three.js library. The page features a canvas element for rendering the 3D scene, with interactive controls for navigating through the virtual space. It includes a room with walls, a floor, and a ceiling, and multiple colorful spherical objects resembling planets that move and rotate within the room. The planets are positioned randomly and can be clicked to change their colors. Additionally, there is a graphical user interface (GUI) integrated using dat.gui to control various parameters, such as fog settings, lighting, and the size of the planets.
